## 서울시 맛집 추천 챗봇 구현 프로젝트

### 프로젝트 목표
#### 서울시의 구별 맛집을 알려주고 메뉴, 주차여부, 인접시설, 주소 등을 알려주는 챗봇 구현

### 프로젝트 과정
1) 데이터 확보 - 완료
2) 데이터 전처리 - 완료
3) 챗봇 모델(transformer) 구현 - 완료
4) 배포(텔레그램) - 완료
